# Статистический анализ данных

Программа для анализа экспериментальных данных с GUI интерфейсом. Оценивает параметры распределений, проверяет статистические гипотезы и строит графики.

## Возможности

**Оценка параметров распределений:**
- Нормальное распределение и распределение Вейбулла
- MLE (метод максимального правдоподобия) и MLS (метод наименьших квадратов по Агамирову)
- Работа с цензурированными данными

**Статистические критерии:**
- Критерий Граббса - обнаружение выбросов
- Критерий Фишера - сравнение дисперсий
- Критерий Стьюдента - сравнение средних значений
- ANOVA - сравнение нескольких групп
- Критерий Шапиро-Уилка - проверка нормальности
- Критерий Уилкоксона - непараметрическое сравнение

**Доверительные интервалы:**
- Для среднего, дисперсии и процентилей

**Визуализация:**
- Графики распределений и доверительных интервалов
- Результаты статистических критериев

## Основные методы

**MLE (Maximum Likelihood)** - находит параметры распределения, которые максимально правдоподобны для наблюдаемых данных.

**MLS (Least Squares)** - метод Агамирова, использует линеаризацию и медианные ранги. Эффективен для цензурированных данных.

**Доверительные интервалы** - диапазон значений, в котором с заданной вероятностью (обычно 95%) находится истинное значение параметра.

**Критерии проверки гипотез** - математические методы для проверки предположений о данных (равенство средних, наличие выбросов, тип распределения).

- C++17, Boost Math
- Python (NumPy, Matplotlib, SciPy)
- Qt 6.9.3 для GUI
- Метод Нелдера-Мида для оптимизации
